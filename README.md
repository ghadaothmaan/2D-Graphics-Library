# 2D-Graphics-Library

Implementation of the following graphics drawing algorithms:

..*Line algorithms

....DDA

....Midpoint 


..*Circle algorithms

....Polar 

....Iterative polar

....Midpoint 


..*Curves algorithms

....Bezier curve

....Hermite curve

....Second degree curve

....Splines

..*Clipping algorithms

....Point clipping on a rectangle window

....Line clipping on a rectangle window

....Point clipping on a cricular window

....Line clipping on a circular window

..*Filling algorithms

....Convex filling
